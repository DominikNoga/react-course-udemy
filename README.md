# What is this?
This a repository where i am storing notes and projects from the most popular React.js udemy course: <br>
https://www.udemy.com/course/react-the-complete-guide-incl-redux/

## Repo organization
Each project covers one course module, sometimes there are more projects for one module.
Every module contains file called 'module-name.notes.md', which stores my notes about crucial concepts of React.
With appropriate code examples.

## How to work with this?
If you'd like to use this, you can just clone this repostitory.
And start the project which you would like to try out.
1. Enter the project folder
2. Install the dependencies using 'npm i' command
3. run 'npm run dev' command
